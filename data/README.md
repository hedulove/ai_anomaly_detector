# Kaggle Data Files

Download from: https://www.kaggle.com/competitions/store-sales-time-series-forecasting/data

Place these CSV files in **this folder** (`data/`):

| File | Required |
|------|----------|
| `train.csv` | **Yes** |
| `stores.csv` | **Yes** |
| `holidays_events.csv` | **Yes** (Ecuador holiday logic) |
| `oil.csv` | Recommended (root-cause context) |
| `transactions.csv` | Optional (required for transaction-count preset questions; use `transactions_enriched` via agent) |

You do **not** need `test.csv` or `sample_submission.csv` for this demo.
